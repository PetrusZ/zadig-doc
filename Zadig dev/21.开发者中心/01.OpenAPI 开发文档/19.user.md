## 列出用户信息

**请求**

```
GET /openapi/users
```

**成功返回说明**

| 参数名     | 类型                    | 描述     |
| ---------- | ----------------------- | -------- |
| `users` | [][User](#user-1) | 用户列表 |
| `total`    | int                     | 用户总数 |

<h4 id="user-1">User 参数说明</h4>

| 参数名          | 说明                                                         | 类型   |
| --------------- | ------------------------------------------------------------ | ------ |
| `uid`      |   uid                                                   | string |
| `name`     | 用户名                                                     | string |
| `accout`   | 账户名                                                         | string |

**成功返回示例**

```json
{
    "users": [
        {
            "uid": "ddd405d5-5131-11ee-b458-4a4088364d94",
            "account": "admin",
            "name": "admin"        
        },
        {
            "uid": "d456d705-73a9-11ee-98cf-56ef622fc735",
            "account": "demo",
            "name": "demo"            
        }        
    ],
    "total_count": 2
}
```
